## 🌟 About the Project

The Tutor Support System (HCMUT_TSS) is a platform built for Ho Chi Minh City University of Technology to connect students with tutors for academic advising and soft skill development[cite: 1]. This repository contains the **Frontend** implementation, providing a responsive and interactive user interface for the system.

## 🚀 My Contributions

As a **Frontend Developer**, I have focused on:
*   **UI/UX Implementation:** Building the interface based on the system design, ensuring a seamless experience for both students and tutors.
*   **Feature Integration:** Developing core client-side functionalities including session booking, personal profile management, and participation monitoring[cite: 1].
*   **MVP Development:** Actively participating in building the Minimum Viable Product (MVP) to validate key user workflows[cite: 1].

## 🛠️ Technology Stack

This project is built as a modern, high-performance Single Page Application (SPA) using the following stack:

*   **Core Framework:** [React 19](https://react.dev/) for component-based UI development.
*   **Routing:** [React Router DOM 7](https://reactrouter.com/) for dynamic client-side navigation.
*   **Styling:** [Tailwind CSS 3.4](https://tailwindcss.com/) for utility-first, responsive design, powered by PostCSS and Autoprefixer.
*   **Build Tool:** [Vite 7](https://vitejs.dev/) for optimized bundling and a fast development experience.
*   **Icons:** [Lucide React](https://lucide.dev/) for clean and consistent iconography.
*   **Code Quality:** Enforced with ESLint 9, including support for React hooks and Fast Refresh.

## 💡 Key Features

*   **Interactive Scheduling:** Students can browse available tutor slots and manage their upcoming sessions[cite: 1].
*   **Progress Tracking:** Tutors can record session notes, and students can view their learning progress trends[cite: 1].
*   **Feedback System:** Built-in forms for students to submit session feedback, which is aggregated for departmental reports[cite: 1].
*   **Dynamic Dashboards:** Administrative views for monitoring participation results and generating academic overview reports[cite: 1].
